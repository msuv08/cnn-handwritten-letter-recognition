## Deployment for TTF_to_PNG.py

This script works as follows:

```python
python3 ttf_to_png.py "relative/path/to/font" size
```
Here's an example of it being used:

```python
python3 ttf_to_png.py "data/tffs/P1-Regular.ttf" 28   
```
